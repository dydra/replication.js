what does this do?

this is an interactive demonstration of data replication through a graph store.

  1. display a known node by selective an identifier from the list to the upper right
  2. drag the node
  3. all browsers which display that node on this page will move the node to the new position

this involves sending a patch request for each movement to sparql graph store service, storing the patch request there and replicating the request to all client browsers which display this page.
